# Brief description of the Activity: 
  This laboratory exercise focuses on performing Data Definition Language (DDL) and Data Manipulation Language (DML) operations within a MySQL environment. It guides the user through creating a table, modifying its structure using ALTER and RENAME commands, and populating it with student records. Finally, the exercise demonstrates advanced data retrieval techniques, including the use of string functions, date formatting, and aggregate functions to organize and present student information effectively.
